
# Windows Kurzbefehle

## Allgemein

| Tastenkombination | Funktion |
|------------------|----------|
| Ctrl + C | Kopieren |
| Ctrl + V | Einfügen |
| Ctrl + X | Ausschneiden |
| Ctrl + Z | Rückgängig |
| Ctrl + Y | Wiederholen |
| Ctrl + F | Suchen |

---

## Windows

| Tastenkombination | Funktion |
|------------------|----------|
| Win + R | Ausführen |
| Win + E | Explorer öffnen |
| Win + L | Computer sperren |
| Win + D | Desktop anzeigen |
| Alt + Tab | Zwischen offenen Fenstern wechseln |
| Shift + Herunterfahren | Vollständiges Herunterfahren (kein Schnellstart) |

---

## Nützliche Befehle

| Befehl | Funktion |
|---------|----------|
| winver | Windows-Version anzeigen |
| gpupdate /force | Gruppenrichtlinien aktualisieren |
| shutdown -p | Computer sofort vollständig herunterfahren |
